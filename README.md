# vt_inet4031_puppet_lab11

## Details
All of the above files serve as a manifest for creating a LAMP stack server. This is a configuration as code way to enable a creation of a web server. Everything here is automated and done through code. All files are ran through puppet (.pp) while the php is done through another file that is connected to the puppet manifest file. 

## Files (testing_puppet and server_users_groups) 
Testing puppet is a file that is seperate from the manifest file. This serves as a test of puppet working. The server and users group file can be configured by user to change and add users. The users are added through a naming standard and can be put into groups if wished to.
